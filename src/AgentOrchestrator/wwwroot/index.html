Agent-to-Agent Communication Lab

Welcome to the .NET 10 Agent Lab

This demo shows a .NET 10 Agent built on the Microsoft 365 Agents SDK with Semantic Kernel orchestration. M365-related queries are delegated to the Copilot Chat API.

Try asking:

  • "Summarize my emails from today"
  • "What meetings do I have tomorrow?"
  • "Find documents about the project proposal"
  • "Who is an expert in Azure in my team?"
  • "Explain what REST APIs are" (general knowledge)
  • "Summarize my emails and explain microservices" (multi-intent)
